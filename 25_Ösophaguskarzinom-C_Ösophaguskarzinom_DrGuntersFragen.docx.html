Ösophaguskarzinom:

→ Haben Sie Schluckstörungen bemerkt? 

→ Haben Sie Probleme beim Schlucken?

→ Treten Schluckbeschwerden bei fester Nahrung auf oder auch bei flüssiger Nahrung? 

→ Haben Sie das Gefühl, dass die Nahrung/die Flüssigkeit im Hals stecken bleibt? 

→ Kommt Ihnen die Nahrung wieder hoch? 

→ Seit wann haben Sie Schluckbeschwerden?

→ Haben die Beschwerden plötzlich oder langsam begonnen? 

→ Sind die Beschwerden mit der Zeit besser oder schlimmer geworden?

→ Haben Sie auch Schmerzen beim Schlucken bemerkt?

→ Hat sich die Farbe Ihres Stuhls verändert? 

→ Wie sieht der Stuhlgang aus?

→ Haben Sie Blut im Stuhl bemerkt? 

→ Ist/war Ihnen übel? → Mussten Sie erbrechen? 

→ Wie sah das Erbrochene aus? Haben Sie Nahrungsreste oder Blutspuren gesehen? 

→ Welche Farbe hatte das Blut?

→ Haben Sie Sodbrennen? -Reflux

→ Verspuren Sie Brennen hinter dem Brustbein?

→ Würden Sie sagen, dass Sie ein Druckgefühl hinter dem Brustbein haben? 

→ Müssen Sie häufig sauer aufstoßen? 

→ Haben Sie ein Völlegefühl?

→ Wann leiden Sie unter diesen Symptomen?

→ Nach Mahlzeiten oder wenn Sie nüchtern sind? 

→ Wie ernähren Sie sich? (Essen Sie viel fettig, scharf oder ernähren Sie sich eher gesund?) 

→ Essen Sie die Speise sehr heißt oder eher kalt?

→ Essen Sie gerne Wurst oder Salami?
